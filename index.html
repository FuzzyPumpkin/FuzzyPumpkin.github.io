Colby Critiques

Tools used:

  • HTML/CSS/Javascript
  • Node.js
  • MongoDB
  • Bootstrap 4.0
View on GitHub

This is a small, very specialized review/blog site created with Node.js, MongoDB, and various packages, using Bootstrap 4.0 as a CSS framework. The site features user authentication and permissions relating to the edit/create/delete routes. Bootstrap 4.0 was used to show familiarity with the framework and styles were kept mostly to those available with the framework. I tried to keep manual CSS to a minimum since manual CSS is my preference, as a challenge.

Future plans:

  • Design overhaul. Right now the site look is very basic.
  • Adding subtle animations.
  • Splash/landing page. It would add to the experience to have a small intro to Colby and his world.

As an added bonus, there is an easter egg you can find on the site. You must be logged in. Happy clicking!

Stacey Evy: Realtor

Tools used:

  • HTML/CSS/Javascript



View on GitHub

This is a site created for a fictious realtor. I gave myself a challenge of creating a fully-functional static site that could be used on a professional level within five days (approximately 8.5 working hours) as my first focused complete-site project. I was able to achieve that goal. There are a few subtle CSS and JavaScript animations on the site, as well as custom JavaScript form validation.

Site passes all Lighthouse audits with 90% or above. The sticking point is on the index page's accessibility test, as for some reason that particular page's test incorrectly marks the text on the parallax footer as being on a white background. Without that error, it should be at 100% accessibility similar to the other two pages. Site is fully mobile-friendly and tested on such, though it displays well on standard desktop and wide-screen monitors.
